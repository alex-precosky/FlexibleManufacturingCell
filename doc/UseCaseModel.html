 V2004.1
 Significant overhaul to reflect more knowledge about system revealed by simulator.  Significant changes are the removal of the use cases for start-up, shut-down, or emergency situation.  System no longer has "modes".  Control system startup is now specified in the process specification.  Actions given their names they will take on as C functions.
