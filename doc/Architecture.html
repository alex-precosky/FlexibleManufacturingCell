The system was designed to be implemented as a single node.  All actuators and sensors have control lines running into the control station.  Directly interfacing with the hardware sensors and actuators is the layer specified in flexInterface.h.  The high-level design documented in these documents is implemented in the top layer, the control software, which consists of a series of POSIX threads and structs.
